# 🎓 Student Performance Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze student data and provide meaningful insights into academic performance, attendance, fee collection, and demographic distribution.

The dashboard helps educational institutions monitor important KPIs, identify trends, and make data-driven decisions for improving student performance and overall management.

---

## 📊 Key Features

- Interactive Dashboard
- Dynamic Filters (Slicers)
- KPI Cards
- Student Performance Analysis
- Attendance Monitoring
- Fee Collection Analysis
- Gender-wise Distribution
- Course-wise Insights
- Department-wise Analysis
- Data-driven Visualizations

---
